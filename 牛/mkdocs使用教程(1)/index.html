mkdocs使用教程

1.新建一个项目文件夹，然后再终端命令行中输入

git clone https://github.com/hasongo/mkdocs.git 克隆仓库 (只有第一次需要克隆，后续再添加文件输入git pull)

  1. 到doce文件夹下创建对应的文件夹放入md文件

  2. 回到上级目录然后再命令行中输入 git add .

  3. 再命令行中输入 git commit -m '改变的提示'

  4. 再命令行中输入 git push 推送到远程仓库

  5. 在mkdocs.yml同级目录下输入mkdocs gh-deploy
